2.4 配置我们的博客

网站的设置大部分都在 _config.yml 文件中，详细配置可以查看官方文档

下面只列出简单常用配置

  • title -> 网站标题
  • subtitle -> 网站副标题
  • description -> 网站描述
  • author -> 您的名字
  • language -> 网站使用的语言

坑：进行配置时，需要在冒号:后加一个英文空格
